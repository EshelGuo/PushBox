推箱子是一款经典的电子游戏，要求玩家在二维地图上把箱子推到指定地点，当中牵涉到大量的空间逻辑推理。
HTML5 Sokoban 是一个非常不错的在线推箱子的网页。

Sokoban官网: http://sokoban.ws/sokoplayer/SokoPlayer_HTML5.php

推箱子关卡一般用XSB格式来保存和交流，解答步骤则使用LURD格式，请参见：XSB和LURD格式简介:

XSB格式规定使用以下符号：

@ ==> 人 man
+ ==> 人在目标点 man on goal
$ ==> 箱子 box
* ==> 箱子在目标点 box on goal
# ==> 墙 wall
. ==> 目标点 goal
- ==> 地板 floor，还可以使用空格或者下划线表示。

LURD格式规定使用以下八个拉丁字母(小写字母表示移动，大写字母表示推动)：

l  或 L ==> 左 Left
r 或 R ==> 右 Right
u 或 U ==> 上 Up
d 或 D ==> 下 Down
